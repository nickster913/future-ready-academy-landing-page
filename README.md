# AI Genesis Blueprints

A curated library of production-ready reference architectures, templates, and system patterns for building and operating AI-driven software systems end to end.

This repo is built around **problem-first, stack-second** design and focuses on reliability, observability, and real-world constraints (cost, latency, failure modes).

## Structure
- `blueprints/` End-to-end reference architectures
- `templates/` Project starters
- `modules/` Composable building blocks
- `stacks/` Known-good combinations
- `cookbooks/` Copy/paste playbooks
- `examples/` Minimal runnable demos
- `playgrounds/` Experiments and spikes
- `docs/` Standards, decision notes, diagrams
