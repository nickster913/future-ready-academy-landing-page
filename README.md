# Future Ready Academy — Landing Page

A single-page marketing site for Future Ready Academy's VEX IQ Robotics Bootcamp, a 2-day intensive programme for ages 10–12 held at Fusionopolis, One-North, Singapore.

## About the page

The landing page promotes the bootcamp with a focus on DSA (Direct School Admission) portfolio building. Key selling points include:

- **Carnegie Mellon Robotics Academy certification** — Future Ready Academy is the only CMRA-authorised partner in Singapore and Indonesia
- **VEX IQ robotics** — hands-on building and programming across both days
- **DSA positioning** — credentials verified by SST, Hwa Chong, Raffles, and other MOE schools with Robotics DSA tracks
- **Small cohort** — capped at 12 seats per bootcamp
